# The Master's Archive - Landing Page

Dự án Landing Page dành cho **The Master's Archive**, một tổ chức với hơn 30 năm kinh nghiệm trong lĩnh vực tư vấn đào tạo và du học nghề chuẩn Đức.

## 🌟 Giới thiệu

Đây là giao diện trang web giới thiệu các dịch vụ giáo dục và hướng nghiệp tại Đức. Mục tiêu của dự án là thiết kế một trang Landing Page chuyên nghiệp, mang đậm phong cách và tiêu chuẩn Đức nhằm kết nối sinh viên / người lao động Việt Nam với các trường học và đối tác tại Châu Âu.

## 🚀 Công nghệ sử dụng
- **HTML5** (Semantic Web)
- **Tailwind CSS** (Utility-first framework để thiết kế giao diện nhanh chóng & hiện đại)

## 📂 Cấu trúc dự án
- `Landing_Page_Desktop_No_Italics.html`: Source code giao diện hiển thị chính.
- `Landing_Page_Desktop_No_Italics.png`: File hình ảnh screenshot của thiết kế giao diện gốc trên Desktop.

## 🌍 Hướng dẫn sử dụng
Để xem trực tiếp website:
1. Tải toàn bộ mã nguồn về máy.
2. Mở trực tiếp file `Landing_Page_Desktop_No_Italics.html` bằng bất kỳ trình duyệt web nào (Google Chrome, Edge, Safari, v.v.).
